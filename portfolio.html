Single Page Application using GitHub API


This is a SPA that I have created during my Hack Your Future class. First a user can select a repository from the drop down list, then the application displays high-level information about different repositories available at Hack Your Future. When clicking on the repository name or one of the contributors to that repository, the user will be taken to that repository or contributor's information in a new browser tab. I began this SPA with basic callbacks, then refactored it to include Promises, and then added ES6 Classes and used async/await syntax. Finally I have made the app ARIA-compliant and made it media-responsive.
